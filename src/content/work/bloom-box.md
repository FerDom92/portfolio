## Role Overview

As a Frontend Developer at Frávega Tech, I focused on maintaining and developing modern web applications for one of Argentina's leading retail companies.

## Key Responsibilities

- Maintained and developed various user interfaces using Next.js, React, and Vitest
- Updated legacy systems to modern architectures and technologies
- Worked closely with the QA team to prevent potential bugs before deployment
- Deployed new features across different environments for comprehensive testing
- Developed unit and end-to-end tests exceeding quality gate standards
- Updated old components to the new UI library following strict design guidelines
- Collaborated with the backend team to develop new features and integrations
- Maintained up-to-date Jira tickets for proper task traceability

## Technologies Used

Next.js, React, TypeScript, Vitest, Jira, Git

## Impact

Successfully modernized multiple legacy systems, improving code quality and test coverage across the platform.
