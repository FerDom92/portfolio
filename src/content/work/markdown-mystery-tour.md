## Role Overview

At Natural Tech House, I worked on multiple high-profile projects including institutional websites for Grupo Petersen and various CMS and API solutions.

## Key Projects

### Institutional Websites for Grupo Petersen
- Built modern, scalable websites using Next.js, React, and Contentful CMS
- Implemented comprehensive testing with Jest
- Automated deployments using GitLab CI/CD

### Backoffice CMS Management
- Developed a complete backoffice system to manage and automate form information
- Built with React.js, Node.js, Express.js, and MongoDB
- Streamlined content management workflows

### BFF API Development
- Created a unified API applying the Backend-for-Frontend (BFF) pattern
- Implemented comprehensive API documentation with Swagger
- Developed automated testing suite with Postman
- Built with Node.js and Express.js

### Infrastructure & DevOps
- Designed and implemented infrastructure to automate CMS component catalog deployment
- Utilized GitLab CI, StoryBook, Vercel, and Docker
- Established CI/CD pipelines for consistent deployments

### Medical Services Management System
- Developed a comprehensive system for managing medical services
- Used Next.js for frontend and Nest.js for backend
- Integrated modern architecture patterns

## Technologies Used

Next.js, React, Node.js, Express.js, Nest.js, MongoDB, Contentful, Jest, GitLab CI, Vercel, Docker, StoryBook, Swagger, Postman

## Impact

Successfully delivered multiple enterprise-level applications improving operational efficiency and user experience for major clients.
